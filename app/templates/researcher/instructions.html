Instructions for the researcher

This is the instruction page for the researcher on how to use this video annotation web-app. First, we will introduce the different features of the app in the 'Functionalities' section and inform all the known bugs and limitations of the app in the 'Limitations' section. If you have any issues with the app or you want to report a bug, please contact us via slack or e-mail.

Functionalities

Configure page: The configuarion page, where the researcher can add or remove videos or change the database this is connected to.

Raw data page: On this page the researcher can view the raw user input data with columns "videoid", "username", "timestamp", "value" (valence), "value2" (intensity of the valence) and "date". On the upper left corner there is a "Delete all" button for deleting all the data. On the upper right corner there is a "Export" button for exporting the data.

Clusters page: Display correlation plot for the researcher. This webpage is only for the role researcher.

Chart page: This page will show the Valence ratings by timestamp -plot for a chosen video, where Valence rating is on the y-axis and Timestamp (in seconds) is on the x-axis. The researcher can change the current plot view to the plot of another video by selecting the name of that video from the 'Video' button on the upper left corner. The heading above the chart will inform to which video the specific plot belongs to.

Change role: From this page the researcher can change the role between 'User' and 'Researcher' but also choose the role 'Test/Both' for debugging purposes.

Limitations

Slow loading with chart view: The chart view loads in 4.9 seconds currently. This bug is linked to rendering the plot and calculating the values that the plot uses in server side. The amount of data points will probably affect by increasing the loading time of the chart view. Updated April 23rd 2019.
